# Pomo

> Your sanctuary for deep work.

A beautifully designed Pomodoro timer that creates calm, not anxiety.

## Features

- Accurate timer with Web Worker support
- "The Breath" - 3-second breathing animation before focus sessions
- Elegant completion celebrations
- Dark/Light mode with system detection
- Keyboard-first interaction
- PWA support (install on any device)
- No ads, no gamification, no guilt

## Getting Started

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build
```

## Tech Stack

- Next.js 14 (App Router)
- Tailwind CSS
- Framer Motion
- TypeScript

## Philosophy

Pomo is built on the principle of **quiet confidence**. While other timer apps race to add features, gamify productivity, and create anxiety with streaks, we subtract.

Every interaction is designed to reduce stress, not add it.

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `Space` | Start / Pause |
| `R` | Reset |
| `S` | Skip to next session |
| `D` | Toggle dark mode |
